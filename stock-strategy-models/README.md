# A股系统化交易策略模型

> 本项目把一套适合普通投资者学习和复盘的 A 股策略框架整理成可运行的 Python 项目。
>
> 重点不是预测明天涨跌，而是建立固定模型、仓位纪律和风险控制。

## 项目包含的策略

### 1. 趋势龙头模型

核心思想：

- 只做市场主线
- 只做板块中相对强势的标的
- 用成交量、趋势和相对强度过滤后排杂毛

适合场景：

- 科技成长、AI、半导体、机器人、新能源等主线行情
- 指数放量、风险偏好上升阶段

### 2. EMA 趋势系统

核心思想：

- 股价站上 20EMA
- 20EMA 高于 60EMA
- 趋势确认后才参与
- 跌破趋势线后减仓或退出

优点：

- 简单
- 可回测
- 能过滤大量下跌趋势股票

### 3. 机构价值趋势模型

核心思想：

- 好公司
- 好行业
- 好价格
- 趋势确认后再介入

观察指标：

- ROE
- 毛利率
- 经营现金流
- 行业空间
- 估值位置

### 4. 大盘择时与仓位模型

核心思想：

- 牛市提高仓位
- 震荡市降低仓位
- 熊市或指数跌破关键均线时空仓或轻仓

示例规则：

- 指数 > 20 日均线：允许做多
- 指数 < 60 日均线：降低仓位
- 市场成交额低迷：降低交易频率

## 风险控制原则

1. 不满仓梭哈
2. 不追高潮日后排股
3. 不抄下跌趋势
4. 单笔亏损提前设定上限
5. 盈亏比优先于胜率
6. 空仓也是一种仓位
